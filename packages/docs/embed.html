Embedding a .boxes file

Render any .boxes graph in a plain HTML page using only the Cytoscape.js CDN — no build tools, no npm, no framework required.

How to embed

A .boxes file is plain JSON. The three fields Cytoscape.js needs are elements, userStylesheet, and lastLayout. The entire embed fits in < 30 lines of HTML.

1

Add Cytoscape.js from the CDN

No install or build step needed — a single <script> tag is enough.

<script src="https://unpkg.com/cytoscape@3/dist/cytoscape.min.js"></script>
2

Add a container element

Give it explicit dimensions — Cytoscape.js measures the container when it initialises.

<div id="graph" style="width: 100%; height: 500px;"></div>
3

Fetch the .boxes file and initialise Cytoscape

elements maps directly to Cytoscape's elements option. userStylesheet is a standard Cytoscape stylesheet array and can be passed as style directly. lastLayout records which layout was last applied; fall back to preset to honour saved node positions. 

fetch('my-graph.boxes')
  .then(r => r.json())
  .then(graph => {
    cytoscape({
      container: document.getElementById('graph'),
      elements: graph.elements,
      style:    graph.userStylesheet,
      layout:   graph.lastLayout ?? { name: 'preset' }
    });
  });
Serving the file: Browsers block fetch() on the file:// protocol. Serve the page from a local web server (e.g. npx serve . or VS Code Live Server) or place the .boxes file on the same origin as your HTML page.

Complete copy-paste example

Drop this into an index.html file alongside your .boxes file and open it with a local server. The graph renders in the browser with no dependencies beyond Cytoscape.js. 

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Graph</title>
  <script src="https://unpkg.com/cytoscape@3/dist/cytoscape.min.js"></script>
  <style>
    body  { margin: 0; background: #f8f9fa; }
    #graph { width: 100vw; height: 100vh; }
  </style>
</head>
<body>
  <div id="graph"></div>

  <script>
    fetch('my-graph.boxes')
      .then(r => r.json())
      .then(graph => {
        cytoscape({
          container: document.getElementById('graph'),
          elements:  graph.elements,
          style:     graph.userStylesheet,
          layout:    graph.lastLayout ?? { name: 'preset' }
        });
      })
      .catch(err => console.error('Failed to load graph:', err));
  </script>
</body>
</html>

About the format

A .boxes file is a UTF-8 JSON document containing an elements object (nodes & edges), a userStylesheet array, a lastLayout descriptor, and optional context and version fields.
