Features

Everything you need to build graph editors

Boxes wraps Cytoscape.js's powerful graph engine with a full-featured editing experience, ready to embed in any web application.

🧩

Framework Support

First-class Vue 3 and React 18 components alongside a vanilla JS core. Consistent API across all integrations.

🎨

CSS Styling as Data

Cytoscape.js stylesheet rules are baked into the graph data model. Style nodes and edges declaratively and export styles with your graph.

📐

10+ Layout Algorithms

Built-in: Random, Grid, Circle, Concentric, Breadth-first, CoSE. Extensions: Dagre, Cola, Klay, Euler, fCoSE, ELK, and more.

🔗

Semantic Web Ready

Export graphs to RDF/Turtle, JSON-LD, and RDF/XML. Import from JSON-LD. Native support for OWL, RDFS, and SKOS ontologies.

↩️

Full Undo/Redo

50-level undo/redo history for all editing operations. Keyboard shortcuts (Ctrl+Z / Ctrl+Y) work out of the box.

✂️

Cut, Copy & Paste

Full clipboard support for nodes and their connected edges, with cascading paste offsets to avoid overlap.

🏷️

Typed Graph Elements

Define node and edge types with labels, default properties, colors, and shapes. Built-in templates for OWL ontologies and Arrows diagrams.

🖱️

Interactive Edge Drawing

Drag from a node handle to draw edges interactively. Context menus for quick node and edge operations.

💾

Flexible I/O

Import/export as Cytoscape JSON, RDF/Turtle, JSON-LD, RDF/XML, Arrows.app JSON, Lucidchart CSV, SVG, and PDF. Extensible I/O plugin system.

Quick Start

Up and running in minutes

1

Install the package

Choose the integration that fits your project.

# Vanilla JS / core library
npm install boxes-core

# Vue 3
npm install boxes-vue

# React 18
npm install boxes-react
2

Create a container element

<div id="graph"
  style="width:800px;height:500px">
</div>
3

Initialize the editor

import { BoxesEditor } from 'boxes-core';

const editor = new BoxesEditor(
  document.getElementById('graph'),
  {
    nodeTypes: [
      { id: 'person', label: 'Person',
        color: '#4a90d9', shape: 'ellipse',
        data: { name: '' } }
    ],
    edgeTypes: [
      { id: 'knows', label: 'knows',
        color: '#666' }
    ]
  }
);

// Listen for changes
editor.on('change', (graph) => {
  console.log(editor.exportGraph());
});

Vue 3

<template>
  <BoxesEditor
    :node-types="nodeTypes"
    :edge-types="edgeTypes"
    @change="onGraphChange"

import { BoxesEditor } from 'boxes-react';

Palette

Customise node & edge types

Pass nodeTypes and edgeTypes to the constructor to populate the palette sidebar. Each type defines a label, colour, shape, and optional default property values that are stamped onto every new element created from that type. 

import { BoxesEditor } from 'boxes-core';

const editor = new BoxesEditor(
  document.getElementById('graph'),
  {
    // ── Node types ─────────────────────────────────
    // `color` and `shape` power the palette icons.
    // To apply those styles to actual graph nodes,
    // add matching stylesheet rules below (see `style`).
    nodeTypes: [
      {
        id: 'class',
        label: 'Class',
        color: '#4a90d9',
        shape: 'ellipse',
        // Stamp a `type` property on every new Class node
        // so stylesheet selectors can target it:
        data: { type: 'class', '@type': 'owl:Class' }
      },
      {
        id: 'instance',
        label: 'Instance',
        color: '#27ae60',
        shape: 'round-rectangle',
        data: { type: 'instance', '@type': 'owl:NamedIndividual' }
      },
      {
        id: 'literal',
        label: 'Literal',
        color: '#e67e22',
        shape: 'rectangle',
        data: { type: 'literal', datatype: 'xsd:string', value: '' }
      },
      {
        id: 'annotation',
        label: 'Annotation',
        color: '#8e44ad',
        shape: 'diamond',
        data: { type: 'annotation' }
      }
    ],

    // ── Edge types ─────────────────────────────────
    edgeTypes: [
      {
        id: 'subclass',
        label: 'subClassOf',
        color: '#4a90d9',
        data: { type: 'subclass', '@type': 'rdfs:subClassOf' }
      },
      {
        id: 'property',
        label: 'hasProperty',
        color: '#27ae60',
        data: { type: 'property', '@type': 'owl:ObjectProperty' }
      },
      {
        id: 'type',
        label: 'rdf:type',
        color: '#e67e22',
        data: { type: 'type', '@type': 'rdf:type' }
      }
    ],

    // ── Stylesheet ─────────────────────────────────
    // Cytoscape.js uses a stylesheet, not inline styles.
    // Match on the `type` data property stamped above.
    style: [
      { selector: 'node[type="class"]',
        style: { 'background-color': '#4a90d9',
                 'shape': 'ellipse' } },
      { selector: 'node[type="instance"]',
        style: { 'background-color': '#27ae60',
                 'shape': 'round-rectangle' } },
      { selector: 'node[type="literal"]',
        style: { 'background-color': '#e67e22',
                 'shape': 'rectangle' } },
      { selector: 'node[type="annotation"]',
        style: { 'background-color': '#8e44ad',
                 'shape': 'diamond' } },
      { selector: 'edge[type="subclass"]',
        style: { 'line-color': '#4a90d9',
                 'target-arrow-color': '#4a90d9' } },
      { selector: 'edge[type="property"]',
        style: { 'line-color': '#27ae60',
                 'target-arrow-color': '#27ae60' } },
      { selector: 'edge[type="type"]',
        style: { 'line-color': '#e67e22',
                 'target-arrow-color': '#e67e22',
                 'line-style': 'dashed' } }
    ]
  }
);

// Switch types programmatically:
editor.setNodeType('instance');
editor.setEdgeType('property');

// Add a node of a specific type at a position:
editor.addNodeOfType('class', { x: 200, y: 150 });

Field Type Description
id string Unique key used in setNodeType()
label string Shown in palette and stamped as the element's label
color CSS colour Palette icon colour only — also add a style rule to apply it to the graph
shape string Palette icon shape only — also add a style rule to apply it to the graph
data object Default properties merged into every new element; use a type field as the stylesheet selector target
style (constructor) array Cytoscape.js stylesheet rules — required to apply colours & shapes to the actual graph
Packages

Choose your integration

All packages share the same underlying Cytoscape.js engine and consistent API.

boxes-core

Core Library

The pure JavaScript graph editor built directly on Cytoscape.js. Use with any framework or no framework at all.

npm install boxes-core
boxes-vue

Vue 3 Component

A Vue 3 single-file component that wraps the core editor. Reactive props, emitted events, and defineExpose access.

npm install boxes-vue
boxes-react

React Component

A React 18 component using forwardRef and useImperativeHandle for full imperative API access.

npm install boxes-react
